    Set Animated Gif as Wallpaper

    Jan 14, 2021
    #linux #tips #tech

    NOTE: Environment Ubuntu 20.04

    Dependencies #

    • Xwinwrap:
    1sudo apt-get install xorg-dev build-essential libx11-dev x11proto-xext-dev libxrender-dev libxext-dev
2git clone https://github.com/ujjwal96/xwinwrap.git
3cd xwinwrap
4make
5sudo make install
6make clean
    • Gifsicle:
    1sudo apt install gifsicle

    The helper script #

    A helper script to setup animated .gif in dual monitors.

     1#!/bin/bash
 2# Uses xwinwrap to display given animated .gif in dual monitors.
 3if [ $# -ne 1 ]; then
 4    echo 1>&2 Usage: $0 image.gif
 5    exit 1
 6fi
 7gif=$1
 8killall -9 xwinwrap
 9killall -9 gifview
10# Get monitors resolution
11SCR1=`xrandr | awk '/primary/ && /connected/ { print $4 }'`
12SCR2=`xrandr | awk '!/primary/ && /connected/ { print $3 }'`
13
14xwinwrap -g $SCR1 -ov -ni -s -nf -- gifview -w WID $gif -a &
15xwinwrap -g $SCR2 -ov -ni -s -nf -- gifview -w WID $gif -a &

    If you want to run xwinwrap by yourself, here is the example:

    1xwinwrap -g 1920x1080 -ov -ni -s -nf -- gifview -w WID /full/path/to/gif -a
