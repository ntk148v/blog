    Rename Files in Linux

    Dec 16, 2021
    #linux #tech
    • Rename a single file with mv. Just a basic thing.
    • Rename multiple files with mv.
    1# Rename files with suffix .yaml to yml
2for f in *.yaml; do mv -- "$f" "${f%.yaml}.yml"
    • Rename multiple files with rename.
     1# Install rename command
 2# Ubuntu/Debian-derived distros
 3sudo apt install rename
 4# RedHat-derived distros
 5sudo yum install prename
 6# The follow examples are performed in Ubuntu/Debian-derived distros
 7rename 's/.yaml/.yml/' *.yaml
 8# Replace all occurrences of "prev_" with "next_"
 9rename 's/prev_/next_' *.c
10# Delete part of a filename
11rename 's/next_//' *.c
12# Limit changes to specific parts of filenames
13# Only change the files that start with "paramater"
14rename 's/^param/parameter/' *.c
15# Search with grouping
16# Replace all occurrences of "string" and "strong"
17rename 's/(str|stro)ng/strength' *.c
18# Use translations with rename
19# Force filenames to uppercase
20rename 'y/a-z/A-Z' *.py
21# More?
22man rename
